### Разработка
Для разработки необходим установленный node.js (c npm), git  
остальные dev зависимости, взятые из package.json установит npm  

```
git clone git@github.com:SemenMilokhin/SportRezerv.git
cd SportRezerv
npm i
npm run dev
```
